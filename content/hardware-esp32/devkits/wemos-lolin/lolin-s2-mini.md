# LOLIN S2 Mini

Mini board con [ESP32-S2](../../socs/esp32-s2.md), pinout D1 Mini compatible.

Docs oficiales: [LOLIN S2 Mini](https://www.wemos.cc/en/latest/s2/s2_mini.html)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S2](../../socs/esp32-s2.md) |
| Flash/PSRAM | 4 MB / 2 MB |
| USB | USB-C OTG nativo |
| Form factor | [D1 Mini](../../form-factors.md) (compatible con shields D1 Mini) |
| LiPo | ❌ |
