# SparkFun Thing Plus ESP32-S3

Feather-compatible con [ESP32-S3](../../socs/esp32-s3.md) + Qwiic onboard.

Docs oficiales: [SparkFun Thing Plus ESP32-S3](https://www.sparkfun.com/sparkfun-thing-plus-esp32-s3.html)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 4 MB / 2 MB |
| USB | USB-C OTG nativo |
| Form factor | [Feather](../../form-factors.md)-compatible |
| LiPo | ✅ |
| Qwiic | ✅ ([STEMMA QT](../../../electronica/conectores.md) compatible) |
| Wireless | WiFi 4 + BLE |
## Cuándo elegirlo

- Necesitás [Feather](../../form-factors.md) form factor con S3 + Qwiic integrado
- Específicamente cuando se necesita Qwiic onboard en un Feather con S3
