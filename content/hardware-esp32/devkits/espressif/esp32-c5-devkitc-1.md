# ESP32-C5-DevKitC-1

DevKit oficial para [ESP32-C5](../../socs/esp32-c5.md). Único DevKit con WiFi 6 dual-band (2.4 + 5 GHz).

Docs oficiales: [ESP32-C5-DevKitC-1 user guide](https://docs.espressif.com/projects/esp-dev-kits/en/latest/esp32c5/esp32-c5-devkitc-1/user_guide.html)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-C5](../../socs/esp32-c5.md) |
| USB | USB-C x2 (Serial/JTAG nativo + bridge UART) |
| Antena | Layout dual-band 2.4 + 5 GHz |
## Cuándo elegirlo

- Necesidad de conectarse a una red WiFi 6 en 5 GHz exclusivamente
- Entornos saturados de 2.4 GHz donde 5 GHz es la única opción
