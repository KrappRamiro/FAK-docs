# M5StickS3

Stick portátil con [ESP32-S3](../../socs/esp32-s3.md) y display LCD pequeño.

Docs oficiales: [M5StickS3 docs](https://docs.m5stack.com/en/core/StickS3), [M5StickS3 shop](https://shop.m5stack.com/products/m5sticks3-esp32s3-mini-iot-dev-kit)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 8 MB / 8 MB |
| Display | 1.14" LCD |
| Wireless | WiFi 4 + BLE |
| Conector | Grove |
## Cuándo elegirlo

- Wearable / portátil con display pequeño
- Sucesor moderno del M5StickC clásico
