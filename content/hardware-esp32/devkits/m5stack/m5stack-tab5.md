# M5Stack Tab5

Tablet IoT con [ESP32-P4](../../socs/esp32-p4.md) (compute) + [ESP32-C6](../../socs/esp32-c6.md) (wireless) y pantalla 5".

Docs oficiales: [M5Stack Tab5 shop](https://shop.m5stack.com/products/m5stack-tab5-iot-development-kit-esp32-p4)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-P4](../../socs/esp32-p4.md) + [ESP32-C6](../../socs/esp32-c6.md) |
| Flash/PSRAM | 16 MB / 32 MB PSRAM |
| Display | 5.0" IPS touch 1280x720 |
| Cámara | SC2356 2 MP (MIPI-CSI) |
| Wireless | WiFi 6 + 802.15.4 (vía C6) |
## Cuándo elegirlo

- HMI tipo tablet con display grande
- Procesamiento de video (P4 + H.264 hw)
- El único producto off-the-shelf con pantalla 5" y P4
