# Arduino Nano ESP32

Pinout Nano clásico con [ESP32-S3](../../socs/esp32-s3.md) adentro.

Docs oficiales: [Arduino Nano ESP32 store](https://store.arduino.cc/products/nano-esp32), [Arduino Nano ESP32 docs](https://docs.arduino.cc/hardware-esp32/nano-esp32/)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 16 MB / 8 MB |
| USB | USB-C OTG nativo |
| Form factor | Arduino Nano |
## Cuándo elegirlo

- Migración de proyectos Arduino Nano existentes (mismo pinout)
- Integración nativa con Arduino IDE / Arduino Cloud
- 16 MB de flash + 8 MB PSRAM en formato Nano
