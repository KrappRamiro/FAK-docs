# LilyGo T-Deck

Board portátil con [ESP32-S3](../../socs/esp32-s3.md), QWERTY keyboard, trackball y display touch.

Docs oficiales: [T-Deck repo en GitHub](https://github.com/Xinyuan-LilyGO/T-Deck)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 16 MB / 8 MB |
| USB | USB-C OTG nativo |
| LiPo | ✅ |
| Display | 2.8" IPS touch 320x240 |
| Input | QWERTY keyboard + trackball |
| Extras | SD card slot, opcional [LoRa](../../../red/lora.md) |
## Cuándo elegirlo

- Dispositivos portátiles tipo "comunicador" (Meshtastic + LoRa, walkie-talkie digital)
- Demos interactivos con UI completa (teclado + pantalla)
