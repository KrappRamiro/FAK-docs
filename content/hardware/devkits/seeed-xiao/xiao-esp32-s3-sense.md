# XIAO ESP32-S3 Sense

XIAO con [ESP32-S3](../../socs/esp32-s3.md) + cámara OV2640 + micrófono PDM + SD card en formato tiny.

Docs oficiales: [Seeed XIAO ESP32-S3 wiki](https://wiki.seeedstudio.com/xiao_esp32s3_getting_started/)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 8 MB / 8 MB |
| USB | USB-C OTG |
| Form factor | [XIAO](../../form-factors.md) 21x17.5 mm con módulo cámara snap-on |
| LiPo | ✅ |
| Cámara | OV2640 2MP (variantes recientes envían OV3660 3MP) |
| Audio | Mic PDM integrado |
| Extras | SD card slot |
## Cuándo elegirlo

- Cámara de bajo consumo / footprint mínimo
- Aplicaciones de visión + audio compactas
- Cuando el [ESP32-S3-DevKitC-1](../espressif/esp32-s3-devkitc-1.md) es demasiado grande
