# DFRobot FireBeetle 2 ESP32-S3

Board low-power con [ESP32-S3](../../socs/esp32-s3.md), ecosistema Gravity de sensores.

Docs oficiales: [FireBeetle 2 ESP32-S3 wiki](https://wiki.dfrobot.com/SKU_DFR0975_FireBeetle_2_Board_ESP32_S3)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 16 MB / 8 MB |
| USB | USB-C OTG nativo |
| LiPo | ✓ ETA6003 |
| Extras | Conector GDI para display, header de cámara (incluye módulo OV2640 2MP), ecosistema [Gravity](../../conectores.md) |
## Cuándo elegirlo

- Integración con sensores Gravity de DFRobot
- Diseño optimizado para bajo consumo
