# M5Stack Core S3

Unidad encapsulada con [ESP32-S3](../../socs/esp32-s3.md), display touch, cámara y altavoz.

Docs oficiales: [M5Stack Core S3 docs](https://docs.m5stack.com/en/core/CoreS3)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 16 MB / 8 MB |
| Display | 2.0" IPS touch |
| Cámara | GC0308 0.3MP |
| Wireless | WiFi 4 + BLE |
| Conector | Grove + M5Stack stacking |
## Cuándo elegirlo

- Demos all-in-one con display, cámara, batería y stack modular
- Prototipado sin breadboard
