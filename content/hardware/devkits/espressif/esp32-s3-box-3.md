# ESP32-S3-BOX-3

Smart speaker / display all-in-one basado en [ESP32-S3](../../socs/esp32-s3.md).

Docs oficiales: [esp-box repo en GitHub](https://github.com/espressif/esp-box), [Hardware overview for BOX-3](https://github.com/espressif/esp-box/blob/master/docs/hardware_overview/esp32_s3_box_3/hardware_overview_for_box_3.md)

## Specs

| Spec | Valor |
|---|---|
| SoC | [ESP32-S3](../../socs/esp32-s3.md) |
| Flash/PSRAM | 16 MB / 16 MB octal |
| Módulo interno | [WROOM-1](../../modulos/wroom.md) |
| Display | 2.4" IPS touch $320 \times 240$ |
| Audio | 2 micrófonos (codec ES7210) + speaker (codec ES8311) |
| Sensores | IMU ICM-42670 + sensor temp/humedad AHT30 |
| Extras | Headers Pmod para expansión |
| USB | USB-C nativo (serial + JTAG + programación) |
## Cuándo elegirlo

- Referencia de smart speaker / voice + display
- HMI rápidas con touch screen
- Demos de aplicaciones AI on-device
