# LM2596S

Step-down DC-DC ajustable. Convierte tensión mayor a menor sin disipar el exceso como calor. Eficiencia varía con el voltaje de salida: **\~73% @ Vout=3.3V, \~80% @ 5V, \~90% @ 12V** (per [TI datasheet](https://www.ti.com/lit/ds/symlink/lm2596.pdf) secciones 7.5-7.8).

Datasheet del chip: [Texas Instruments LM2596 (PDF)](https://www.ti.com/lit/ds/symlink/lm2596.pdf), [TI LM2596 product page](https://www.ti.com/product/LM2596).

El módulo "LM2596S" del kit incluye el chip + inductor + capacitores + potenciómetro multivuelta en una PCB pequeña.

## Specs del módulo

| Spec                 | Valor                                                                                                                                             |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Input                | 4.5V mínimo (datasheet §7.3), hasta 40V max. Dropout ~1.5V @ 3A (Fig 7-6) - no confundir con el VFB de 1.23V (feedback reference, no es headroom) |
| Output               | 1.2-37V ajustable (potenciómetro on-board)                                                                                                        |
| Corriente            | Hasta 3A (con disipador 2A continuo)                                                                                                              |
| Eficiencia           | **73% @ Vout=3.3V, 80% @ 5V, 90% @ 12V** (TI typ. curves)                                                                                         |
| Frecuencia switching | ~150 kHz                                                                                                                                          |

## Ajuste

1. Conectar input (12V por ejemplo).
2. Sin carga, medir output con multímetro.
3. Girar el potenciómetro multivuelta hasta llegar al voltaje deseado (5V o 3.3V).
4. **Verificar antes de conectar el ESP32** - si lo configurás mal y entran 12V al pin 5V del ESP32, lo quemás.


El LM2596S es conmutado (~80% eficiencia), no lineal, **no disipa calor cerca de los sensores de temperatura**. Propiedad crítica cuando la T del aire es la variable medida: un LDO disipando 7W (típico bajando de 12V→5V@1A) calienta el aire del gabinete y sesga las lecturas de un [SHT45](../../sensores/temperatura-humedad/sht45.md) cercano 2-5 $^\circ\text{C}$ hacia arriba. Sensirion lo declara explícitamente en el [datasheet SHT4x](https://sensirion.com/media/documents/33FD6951/67EB9032/HT_DS_Datasheet_SHT4x_5.pdf) §5.1 "Hardware integration design": *"the sensor should be located in a position where it is exposed to the gas to be measured, but isolated from any heat sources such as nearby electronics or direct sunlight."*
