# 1N5408

Rectificador silicio de 3 A.

Datasheet típico: [Onsemi 1N5408 (PDF)](https://www.onsemi.com/pdf/datasheet/1n5400-d.pdf) (familia 1N5400-1N5408)

## Specs

| Spec | Valor |
|---|---|
| Tipo | Rectificador silicio |
| Corriente max (I_F) | 3 A |
| Tensión inversa (V_R) | 1000V |
| Caída tensión (V_F) | ~1 V |
| Package | DO-201 (más grande que DO-41) |

## Cuándo usarlo

- Líneas de alimentación principal con más de 1.5A
- Flyback en cargas inductivas grandes
- Cuando necesitás holgura de corriente sobre el [1N4007](./1n4007.md)

Para esta corriente, considerar también el Schottky [1N5822](./1n5822.md) si la eficiencia importa.
