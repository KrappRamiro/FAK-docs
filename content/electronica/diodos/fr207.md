# FR207

Diodo "Fast Recovery" de 2 A.

Datasheet típico (familia FR201-FR207): [Rectron FR201-FR207 (PDF)](https://www.rectron.com/public/product_datasheets/fr201-fr207.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | Fast recovery |
| Corriente max ($I_{F(AV)}$) | 2 A @ $T_A = 75\,^\circ\text{C}$ |
| Tensión inversa ($V_{RRM}$) | 1000V |
| Caída tensión ($V_F$) | **1.3 V max @ 2 A** (Rectron Electrical Char) |
| Tiempo de recuperación ($t_{rr}$) | **500 ns max** (FR207/207P) |
| Package | DO-15 |

## Cuándo usarlo

Versión de mayor corriente del [FR107](./fr107.md). Para motores DC grandes con PWM o fuentes conmutadas a mayor potencia.
