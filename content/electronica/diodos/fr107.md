# FR107

Diodo "Fast Recovery" de 1 A - recovery rápido permite uso en switching de alta frecuencia.

Datasheet típico (familia FR101-FR107): [Diotec FR101-FR107 (PDF)](https://www.diotec.com/files/pdf/fr101-fr107.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | Fast recovery |
| Corriente max ($I_{F(AV)}$) | 1 A |
| Tensión inversa ($V_{RRM}$) | 1000V |
| Caída tensión ($V_F$) | **1.3 V max @ 1 A** ([EIC Semi datasheet](http://www.eicsemi.com/datasheet/fr101_7.pdf), Electrical Char) |
| Tiempo de recuperación ($t_{rr}$) | **500 ns max** (test: $I_F=0.5$A, $I_R=1.0$A, $I_{rr}=0.25$A) |
| Package | DO-41 |

## Cuándo usarlo

- **Flyback en PWM rápido** (>1 kHz)
- Motores DC controlados por PWM
- LEDs de cultivo con dimming PWM (1-10 kHz típico)

Para PWM lento o solo on/off, [1N4007](./1n4007.md) alcanza.
