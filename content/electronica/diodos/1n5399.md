# 1N5399

Rectificador silicio de 1.5 A.

Datasheet típico (familia 1N5391-1N5399): [Vishay 1N5391-1N5399 (PDF)](https://www.vishay.com/docs/88514/1n5391.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | Rectificador silicio |
| Corriente max ($I_{F(AV)}$) | 1.5 A |
| Tensión inversa ($V_{RRM}$) | 1000V |
| Caída tensión ($V_F$) | ~0.95 V típ @ 1.5 A, **1.4 V max @ 1.5 A** (Vishay). El "~0.7 V" rule-of-thumb subestima |
| Package | DO-41 (Vishay) / DO-15 (Rectron/Fairchild - el mismo chip en encapsulado distinto) |

## Cuándo usarlo

Cuando el [1N4007](./1n4007.md) queda corto (cargas > 1 A) pero no tanto como para necesitar el [1N5408](./1n5408.md) (3 A). Mismo encapsulado DO-41.
