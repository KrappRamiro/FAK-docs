# S9013

NPN bipolar de propósito general, variante china común.

Datasheet típico (MCC Semi): [MCC S9013 TO-92 (PDF)](https://www.mccsemi.com/pdf/products/S9013(TO-92).pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | NPN |
| Corriente max ($I_C$) | 500 mA |
| Tensión máx $V_{CEO}$ | 25V |
| Hfe (β) | **min 40 @ $I_C = 500$ mA**; 112-300 @ $I_C = 50$ mA (MCC datasheet); bins G/H/I = 112-166 / 144-202 / 190-300 |
| Package | TO-92 |

## Notas

Equivalente funcional al [2N2222](./2n2222.md) para muchos casos. Muy común en kits chinos.

Pinout TO-92 (lado plano): **E B C**, opuesto al BC547. Verificar antes de soldar.
