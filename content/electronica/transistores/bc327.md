# BC327

PNP bipolar, complementario del [BC337](./bc337.md).

Datasheet típico: [BC327 (PDF)](https://www.onsemi.com/pdf/datasheet/bc327-d.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | PNP |
| Corriente max (I_C) | 800 mA |
| Tensión máx Vce | -45V |
| Hfe (β) | 100-630 |
| Package | TO-92 |

## Lógica inversa

HIGH en base = apagado, LOW = encendido. Útil para conmutar el **lado positivo** de una carga (high-side switching) en niveles de corriente medios.
