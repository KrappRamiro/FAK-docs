# S9012

PNP bipolar, complementario del [S9013](./s9013.md).

Datasheet típico (MCC Semi): [MCC S9012 TO-92 (PDF)](https://www.mccsemi.com/pdf/products/S9012(TO-92).pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | PNP |
| Corriente max ($I_C$) | 500 mA |
| Tensión máx $V_{CEO}$ | -25V |
| Hfe (β) | **min 40 @ $I_C = 500$ mA**; 112-300 @ $I_C = 50$ mA (MCC datasheet); bins G/H/I = 112-166 / 144-202 / 190-300 |
| Package | TO-92 |

## Notas

Variante china muy común en kits de hobby. Equivalente funcional al [BC327](./bc327.md) para muchos casos.

Pinout TO-92 (lado plano): **E B C**, opuesto al BC557. Verificar antes de soldar.
