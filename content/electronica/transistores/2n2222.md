# 2N2222

Transistor NPN bipolar de propósito general - el caballo de batalla para conmutar cargas chicas/medianas desde un GPIO.

Datasheet típico (Onsemi): [2N2222A datasheet (PDF)](https://www.onsemi.com/pdf/datasheet/p2n2222a-d.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | NPN |
| Corriente max ($I_C$) | **600 mA continuos** en TO-92 (P2N2222A) / **800 mA continuos** en TO-18 (2N2222A original metálico) |
| Tensión máx $V_{CEO}$ | **40V** (P2N2222A TO-92, [datasheet onsemi](https://www.onsemi.com/pdf/datasheet/p2n2222a-d.pdf)) / **50V** (2N2222A TO-18) / 30V (2N2222 original sin sufijo A) |
| Hfe (β) | 100-300 @ $I_C=150$ mA |
| $V_{BE(sat)}$ | 0.6-1.2 V (datasheet); típico ~0.7 V a baja corriente |
| $V_{CE(sat)}$ | ≤ 0.3 V max @ $I_C=150$ mA |
| $f_T$ | min 300 MHz (switching rápido) |
| Package | TO-92 (P2N2222A) / TO-18 (2N2222A original) |
