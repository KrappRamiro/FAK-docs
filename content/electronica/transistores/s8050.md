# S8050

NPN bipolar de uso general, variante china común.

> Hay variantes con distintas specs según fabricante: **UTC** = 700 mA / 20V Vceo, **onsemi SS8050** = 1.5 A / 25V Vceo. Antes de usar verificar contra el datasheet del marking real.

Datasheets:
- [UTC S8050 (mirror MakerHero)](https://www.makerhero.com/img/files/download/Transistor-S8050-Datasheet.pdf)
- [Onsemi SS8050 (PDF)](https://www.onsemi.com/download/data-sheet/pdf/ss8050-d.pdf)

## Specs - variante UTC (la más común en kits chinos)

| Spec | Valor |
|---|---|
| Tipo | NPN |
| Corriente max ($I_C$) | 700 mA (UTC) / 1.5 A (onsemi SS8050) |
| Tensión máx $V_{CEO}$ | **20V (UTC)** / 25V (onsemi SS8050) |
| Hfe (β) | UTC bins C=120-200, D=160-300, E=280-400. Onsemi B=85-160, C=120-200, D=160-300 |
| Package | TO-92 |

## Notas

Buen reemplazo del [2N2222](./2n2222.md) cuando se necesita un poco más de corriente. Su complementario PNP es el S8550.

Pinout TO-92 (lado plano): **E B C**. Verificar antes de soldar.
