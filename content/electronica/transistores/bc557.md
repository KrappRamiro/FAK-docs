# BC557

PNP bipolar, complementario del [BC547](./bc547.md).

Datasheet típico: [BC557 (PDF)](https://www.onsemi.com/pdf/datasheet/bc556-d.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | PNP |
| Corriente max (I_C) | 100 mA |
| Tensión máx Vce | -45V |
| Hfe (β) | 110-800 |
| Package | TO-92 |

## Lógica inversa

A diferencia del NPN, en PNP **HIGH en base apaga el transistor** y LOW lo enciende. La corriente fluye de Emisor a Colector.

Útil para conmutar el **lado positivo** de una carga (high-side switching), aunque para esto los MOSFETs P-channel suelen ser preferibles.
