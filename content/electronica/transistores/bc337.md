# BC337

NPN bipolar de propósito general, mayor corriente que BC547.

Datasheet típico: [BC337 (PDF)](https://www.onsemi.com/pdf/datasheet/bc337-d.pdf)

## Specs

| Spec | Valor |
|---|---|
| Tipo | NPN |
| Corriente max (I_C) | 800 mA |
| Tensión máx Vce | 45V |
| Hfe (β) | 100-630 |
| Package | TO-92 |

## Pinout (lado plano hacia vos)

```
 C B E
```

## Casos de uso típicos

- Relays medianos
- Driver de bobinas hasta ~600 mA continuos
- Cargas que superan el límite del [BC547](./bc547.md) o [2N2222](./2n2222.md)
